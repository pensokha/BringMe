# BringMe
Shopping List for practicing knowledge

<h2>Features:</h2>
1.Create grocery</br>
2.Edit grocery</br>
3.View grocery</br>
4.Create shop item</br>
5.Edit shop item</br>
6.View shop items which is in certain grocery</br>
7.Delete shop item</br>

<h2>Core Technology:</h2>
1.Core Data </br>
2.UIKit UITableView/UICollectionView </br>
3.AutoLayout </br>
